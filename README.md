# Vehicle Rental System - Octalogic-test

This project is part of the Octalogic test.

# Tech Stack

- **Backend**: Node.js, Express.js
- **Frontend**: React.js
- **Database**: PostgreSQL

# Database Setup

### 1. Run Migration

```bash
cd backend
npx sequelize-cli db:migrate
```

### 2. Run Seed

```bash
cd backend
npx sequelize-cli db:seed:all
```
